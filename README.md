# blog

Projet: Blog d'analyses Opubliq
Vision stratégique:
Opubliq veut développer 3 axes complémentaires:

Services de consultation actuels (revenus)
Panel social avec Nadjim (long terme, nécessite financement)
Ce blog = établir crédibilité + devenir source de recrutement pour le panel

Objectif du blog:

Publier analyses originales de données publiques (CES, EQC, Vividata, etc.)
Démontrer expertise d'Opubliq au grand public
Créer du contenu partageable (médias, réseaux sociaux)
Éventuellement: transformer lecteurs en panélistes

Phase 1 (maintenant - 6 mois):

Site statique simple sur Netlify (gratuit)
blog.opubliq.com via Namecheap (opubliq.com déjà acheté)
Design professionnel sobre (couleurs Opubliq)
Articles courts analysant données publiques
Pas de comptes utilisateurs encore
Rythme: 2-5 articles/mois

Phase 2 (quand budget/traction):

Migration vers WordPress hébergé
Système de comptes utilisateurs (anti-bots)
Mini-sondages intégrés sur sujets d'actualité
Panélistes peuvent se connecter pour voir analyses + répondre aux sondages
Possiblement monétisation (pubs)

Nom (à décider):
Le Pouls, L'Observatoire, Analyses Opubliq, L'Indice Opubliq 

Stack technique:

Court terme: Netlify + Hugo/Jekyll (Markdown)
Long terme: WordPress auto-hébergé avec plugins pour panel

Lien avec panel social:

Panel = projet distinct avec Nadjim (infrastructure sondages sérieuse)
Blog = vitrine publique + porte d'entrée pour recruter panélistes
Synergies: analyses du blog vont éventuellement utiliser données du panel, panélistes accèdent au blog
